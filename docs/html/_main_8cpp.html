template<class t >
void Randomize ( std::vector< t > &  v)

Use the standard pseudo-random permutation generation algorithm to permute the elements of an std::vector into pseudo-random order with the pseudo-random number generator std::default_random_engine seeded by the Unix epoch time (number of seconds since Jan 1, 1970 UTC, excluding leap seconds). Each permutation is equally likely.
