Pseudo-Random Name Permuter

A tool to permute a list of strings into pseudo-random order in which each permutation is equally likely. Input comes from a hard-coded file Input.txt containing a list of strings, one per line. Blank lines are ignored. Output goes to a hard-coded file Output.txt. The standard pseudo-random permutation generation algorithm is used, based on the pseudo-random number generator std::default_random_engine seeded with the Unix epoch time in seconds.
